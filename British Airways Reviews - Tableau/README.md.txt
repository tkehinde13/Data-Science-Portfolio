# British Airways Review Dashboard

## Overview
This interactive Tableau dashboard visualises customer reviews of British Airways, allowing users to explore average ratings across various parameters. The dashboard offers multiple filters to provide a detailed and intuitive analysis of review trends by country, time, and aircraft type.

## Features
- **Interactive Map**: Select countries to view average customer ratings for different regions.
- **Date Filter**: Analyse ratings by month to identify trends over time.
- **Aircraft Filter**: Drill down into customer ratings based on the type of aircraft.
- **Review Parameters**: Compare ratings for different aspects of the experience, such as service, comfort, food, and entertainment.

## Tools Used
- **Tableau Desktop**: To design and build the dashboard.
- **Excel/CSV**: Data source for customer reviews.

## Key Insights
- Visualise how ratings vary across countries and time periods.
- Identify high-performing and low-performing aircraft types based on customer feedback.
- Spot seasonal trends or recurring issues to improve customer experience.

## Getting Started
1. Download the Tableau workbook file (`.twbx`) from the `Tableau_Workbooks/` folder.
2. Open the file in Tableau Desktop.
3. Interact with the dashboard by applying filters for:
   - **Country**: Click on the map to focus on specific regions.
   - **Date**: Use the dropdown to filter by month.
   - **Aircraft Type**: Select a specific aircraft to analyse its performance.
